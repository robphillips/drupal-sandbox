# Drupal Sandbox

Provides a bare-bones Drupal sandbox with Acquia BLT+PHPCS and additional code tools.

## Getting Started

```shell
git clone git@github.com:robphillips/drupal-sandbox.git 9.x
```

## Drupal Code Repository

```shell
fin run-clone [project] [git-url]
```

## Code Validation

```shell
fin run-validate
```

## Build Code

```shell
fin run-build
```
